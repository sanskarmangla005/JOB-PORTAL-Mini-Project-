# 💼 MERN Job Portal

A full-stack Job Portal web application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js). This platform connects recruiters and job seekers by providing an easy way to post, manage, search, and apply for jobs.

## 🌐 Live Demo

🔗 https://mern-job-portal-8kht.onrender.com

> **Note:** The live demo contains sample job listings added for testing and demonstration purposes.

---

## 🚀 Features

### 👨‍💼 For Job Seekers

* User Registration & Login
* Google Sign-In Authentication
* Browse Available Jobs
* Apply for Jobs
* Manage User Profile

### 🏢 For Recruiters

* Secure Authentication
* Create Job Listings
* Update and Delete Job Posts
* Manage Posted Jobs
* View Applications

### ⚡ General Features

* Google OAuth Authentication
* JWT Authentication & Authorization
* Role-Based Access Control
* Responsive User Interface
* RESTful API Architecture
* MongoDB Database Integration
* Secure Backend APIs

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/sanskarmangla005/JOB-PORTAL-Mini-Project.git
```

### Navigate to Project Directory

```bash
cd JOB-PORTAL-Mini-Project
```

### Install Backend Dependencies

```bash
cd backend
npm install
```

### Install Frontend Dependencies

```bash
cd frontend
npm install
```

### Start Backend Server

```bash
npm start
```

### Start Frontend Development Server

```bash
npm run dev
```
